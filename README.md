Универсальный монитор сетевых подключений с функцией автоматического сбора IP-адресов и доменов отдельно по процессам
Программа помогает быстро понять, на какие внешние серверы обращается конкретное приложение (игра, браузер, мессенджер, VPN-клиент, антивирус и т.д.), когда возникают проблемы с подключением.
Полученные данные можно сразу использовать для настройки любого DPI-bypass инструмента.

Что умеет программа

Захват трафика в реальном времени с выбранного сетевого интерфейса
Фильтрация по процессу (например, только bf6.exe, discord.exe, chrome.exe или любой другой)
Показ имени процесса напротив каждого IP/домена
Режимы отображения: всё / только IP / только домены
Автоматический сбор уникальных IP и доменов за сессию
Экспорт одним кликом:
– чистый список IP (готов к вставке в cust1.txt или любой ipset)
– список доменов
– готовая проверенная стратегия для zapret (с --new в нужных местах)
Автопрокрутка лога + возможность отключить
Работает как в режиме «весь трафик», так и с фильтром по одному процессу

Где это пригодится

Любая онлайн-игра (Battlefield, FC, Apex, Valorant, War Thunder и т.д.)
Мессенджеры и VoIP (Discord, Telegram Desktop, WhatsApp, Zoom)
Браузеры, когда определённые сайты не открываются
VPN/Proxy-клиенты, торрент-клиенты
Любое приложение, которое внезапно перестало подключаться к серверам

Как пользоваться

Запусти Universal DPI Monitor.exe (рекомендуется от имени администратора)
Выбери свой сетевой интерфейс
В поле «Процесс» начни вводить название нужной программы — список подхватится автоматически
(или оставь пустым — будет весь трафик)
Нажми ▶ Старт
Запусти проблемное приложение и воспроизведи проблему (поиск матча, вход в аккаунт, загрузка сайта)
В логе сразу увидишь все внешние IP и домены + имя процесса справа
Нажми кнопку «Экспорт → IP / Домены / Стратегия»
Скопируй нужный блок и добавь в конфиг своего DPI-bypass инструмента
Перезапусти bypass — проблема решена

Имеется готовая стратегия под zapret которую можно добавить. В данном случае исполнено через списки cust1.txt

Приятного и стабильного соединения!
